Armstrong Number

Run python script armstrong_number.py by using the command python armstrong_number.py -n 'Upper limit number'.

usage: armstrong_number.py [-h] [-n UPPERLIMIT]

optional arguments:
  -h, --help            show this help message and exit
  -n UPPERLIMIT, --Upperlimit UPPERLIMIT
                        Input Upper Limit number
